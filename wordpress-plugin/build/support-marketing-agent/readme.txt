== Installation ==

1. Upload the plugin files to `/wp-content/plugins/support-marketing-agent/` or install via WordPress plugin repository
2. Activate the plugin through the 'Plugins' menu
3. Complete the setup wizard or go to **SMA > Settings**
4. Enter your API key from your Support Marketing Agent dashboard
5. Configure chat widget appearance and features
6. Start using shortcodes or blocks to add support features!

= Quick Start =

1. Get your API key from [app.supportmarketingagent.com](https://app.supportmarketingagent.com/settings/api)
2. Add `[sma_ticket_form]` to any page for ticket submission
3. The chat widget automatically appears on all pages
4. Use `[sma_knowledge_base]` for self-service support

== Frequently Asked Questions ==

= Do I need a Support Marketing Agent account? =

Yes, you need a free account to use this plugin. Sign up at [supportmarketingagent.com](https://supportmarketingagent.com).

= Is WooCommerce required? =

No, the plugin works with any WordPress site. WooCommerce features are optional and activate automatically when WooCommerce is detected.

= Can I customize the chat widget? =

Yes! You can customize colors, position, welcome message, and more from the settings page.

= Does it work with my theme? =

Yes, the plugin is designed to work with any WordPress theme. Styles are carefully scoped to avoid conflicts.

= Is the AI feature included? =

AI-powered suggestions are included in all plans. The AI uses your knowledge base to suggest relevant answers.

= How do I add the knowledge base? =

Use the `[sma_knowledge_base]` shortcode or the Gutenberg block. Articles are managed from your Support Marketing Agent dashboard.

= Can I use multiple forms on one page? =

Yes, you can use multiple shortcodes with different configurations on the same page.
