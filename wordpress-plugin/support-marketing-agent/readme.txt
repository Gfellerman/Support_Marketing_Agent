AI-powered helpdesk and email marketing integration. One plugin for support tickets, live chat, and customer engagement.

== Description ==

**Support Marketing Agent** connects your WordPress site to a powerful AI-driven helpdesk and email marketing platform.

= Features =

* **Live Chat Widget** - Floating chat for instant customer support
* **Ticket Submission Forms** - Shortcode and Gutenberg block
* **AI-Powered Responses** - Automatic classification and suggestions
* **WooCommerce Integration** - Order lookup in tickets (coming soon)
* **Knowledge Base** - Self-service support (coming soon)
* **Email Marketing** - Capture and nurture leads (coming soon)

= Quick Setup =

1. Install and activate the plugin
2. Go to Support Agent > Settings
3. Enter your API key from app.supportmarketingagent.com
4. That's it! The chat widget appears automatically

= Shortcodes =

* `[sma_ticket_form]` - Display a ticket submission form
* `[sma_ticket_status]` - Ticket status checker
* `[sma_chat_widget]` - Embed chat inline (optional)

== Installation ==

1. Upload `support-marketing-agent` folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu
3. Configure your API key in Support Agent > Settings

== Frequently Asked Questions ==

= Where do I get an API key? =

Sign up at [supportmarketingagent.com](https://app.supportmarketingagent.com) and find your API key in Settings > API.

= Is WooCommerce required? =

No, but WooCommerce integration provides enhanced features like order lookup.
